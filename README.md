Smart Interview Assistant is a full-stack web application designed to simulate real interview practice for users. It allows users to attempt technical interview questions based on topic and difficulty, get instant evaluation of their answers, and track their performance over time. The system also provides a clean dashboard and summary to help users understand their strengths and weak areas.
This project includes role-based functionality for both users and admin. Users can give interviews and view their own history and performance, while admins can manage interview questions and view overall system data. The project is built using Spring Boot for backend, HTML/CSS/JavaScript for frontend, and MySQL as the database with JWT-based authentication for security.
